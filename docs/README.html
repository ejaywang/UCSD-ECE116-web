ECE 4xx: Embedded Artificial Intelligence Course Materials

This package contains the comprehensive materials for the upper-division ECE course on Embedded Artificial Intelligence. The course is designed for students with prerequisites in Arduino, Python, and Object-Oriented Programming.

Course Documents:

  • embedded_ai_course_design.md: This document outlines the overall course structure, including the course overview, prerequisites, learning objectives, tentative weekly topics, and assessment methods. It also includes a detailed weekly schedule with readings and activities.

  • embedded_ai_assignments.md: This document details the five practical assignments for the course, including lab exercises.

  • assignment_1_student.md: The student-facing document for Assignment 1, outlining tasks, requirements, and submission guidelines.
  • assignment_1_instructor_guide.md: The instructor-facing guide for Assignment 1, including grading rubrics and automation strategies.
  • assignment_1_setup_guide.md: A step-by-step guide for setting up the ESP32-S3 development environment.
  • assignment_1_starter_code/: A directory containing starter code for Assignment 1, including PlatformIO project setup and a Python serial plotter template.

  • assignment_1_model_info.md: Provides details about the pre-trained TensorFlow Lite Micro model used in Assignment 1.

  • equipment_list.md: This document provides a comprehensive list of recommended hardware and software equipment for students.

  • embedded_ai_final_exam.md: This document provides details about the conceptual and design-heavy final exam, including its format, coverage, and sample topics for review.

  • final_exam_questions_pool.md: This document contains a pool of 10 conceptual and design-heavy questions for the final exam, from which the actual exam questions are selected.

  • multiple_choice_questions.md: This document contains a pool of 60 multiple-choice questions with 4 or 5 answer options (including 'check all that apply' and 'None of the above') and varying levels of complexity, suitable for quizzes or exams.
