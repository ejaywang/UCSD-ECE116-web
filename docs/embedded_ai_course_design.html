ECE 4xx: Embedded Artificial Intelligence

Course Overview

This upper-division Electrical and Computer Engineering (ECE) course provides a comprehensive introduction to the principles and practices of Embedded Artificial Intelligence (AI). Students will learn how to develop, deploy, and optimize AI models on resource-constrained embedded systems, such as microcontrollers and edge devices. The course emphasizes hands-on experience, integrating theoretical concepts with practical application through programming assignments and projects. Building upon foundational knowledge in Arduino, Python, and Object-Oriented Programming (OOP), students will explore topics such as TinyML, AI accelerators, on-device inference, and various applications of embedded AI in real-world scenarios.

Prerequisites

  • ECE 2xx: Introduction to Arduino/Microcontrollers: Familiarity with Arduino IDE, basic microcontroller programming (C/C++), digital and analog I/O, and common communication protocols (e.g., I2C, SPI).
  • CS 2xx: Python Programming: Proficiency in Python syntax, data structures, algorithms, and scripting.
  • CS 2xx: Object-Oriented Programming: Understanding of OOP concepts such as classes, objects, inheritance, polymorphism, and encapsulation.

Course Learning Objectives

Upon successful completion of this course, students will be able to:

  • Understand the fundamental concepts and challenges of deploying AI on embedded systems.
  • Select appropriate hardware platforms and software frameworks for embedded AI applications.
  • Develop and optimize machine learning models for resource-constrained environments.
  • Implement embedded AI solutions using Python and C/C++ on microcontrollers and single-board computers.
  • Apply embedded AI techniques to real-world problems in areas such as computer vision, audio processing, and sensor data analysis.
  • Evaluate the trade-offs between model accuracy, computational complexity, power consumption, and memory footprint in embedded AI systems.
  • Design and execute embedded AI projects, from concept to deployment, demonstrating problem-solving and critical thinking skills.
  • Analyze and interpret research papers in the field of embedded AI.

Course Topics (Tentative)

Module 1: Introduction to Embedded AI

  • Definition and Scope of Embedded AI
  • Advantages and Challenges of Edge AI
  • Overview of Embedded Systems Architectures for AI
  • Applications of Embedded AI (e.g., IoT, wearables, robotics, automotive)

Module 2: Fundamentals of Machine Learning for Embedded Systems

  • Brief Review of Machine Learning Concepts (Supervised, Unsupervised, Reinforcement Learning)
  • Neural Networks Basics (Perceptrons, Activation Functions, Backpropagation)
  • Introduction to Deep Learning and Convolutional Neural Networks (CNNs)
  • Data Preprocessing and Feature Engineering for Embedded Devices

Module 3: TinyML and Model Optimization

  • Introduction to TinyML: Machine Learning on Microcontrollers
  • Model Quantization (e.g., 8-bit integer quantization)
  • Model Pruning and Sparsity
  • Knowledge Distillation
  • Efficient Neural Network Architectures (e.g., MobileNet, EfficientNet)

Module 4: Embedded AI Hardware and Software Frameworks

  • Microcontrollers vs. Single-Board Computers for AI (e.g., Arduino, ESP32, Raspberry Pi, NVIDIA Jetson Nano)
  • AI Accelerators (e.g., TPUs, NPUs, DSPs)
  • TensorFlow Lite Micro
  • PyTorch Mobile/Edge
  • OpenCV AI Kit (OAK)

Module 5: On-Device Inference and Learning

  • Deploying Trained Models to Embedded Devices
  • Real-time Inference Techniques
  • Continual Learning and Transfer Learning on the Edge
  • Federated Learning for Distributed Embedded Systems

Module 6: Embedded AI Applications - Computer Vision

  • Image Classification on Edge Devices
  • Object Detection with TinyML Models
  • Image Segmentation
  • Hands-on Projects: Object Recognition with ESP32-CAM, Person Detection with Raspberry Pi and OpenCV

Module 7: Embedded AI Applications - Audio and Sensor Data

  • Keyword Spotting and Voice Commands
  • Anomaly Detection with Sensor Data
  • Time Series Analysis on Embedded Devices
  • Hands-on Projects: Voice Assistant on Arduino, Predictive Maintenance with Sensor Data

Module 8: Security, Privacy, and Ethics in Embedded AI

  • Security Challenges in Edge AI
  • Privacy-Preserving AI Techniques
  • Ethical Considerations for Embedded AI Deployment

Assessment

  • Assignments (80%): Five assignments covering theoretical concepts and programming tasks. Due Friday 7 PM with a 5-hour grace period.
  • Final Exam (20%): A comprehensive final exam covering all course material. Scheduled during Finals Week.
