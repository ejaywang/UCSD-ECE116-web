ECE 4xx: Embedded Artificial Intelligence - Required Equipment List

This document outlines the required hardware and software equipment for students enrolled in the Embedded Artificial Intelligence course. To ensure a consistent and focused learning experience, all assignments will be designed around a single, versatile hardware platform: the ESP32-S3 with Camera Module. This choice provides the necessary capabilities for all course assignments, from basic microcontroller interactions to advanced edge AI applications involving vision, audio, and cloud integration.

Hardware Components

1. ESP32-S3 with Camera Module

  • Recommended Board: Espressif ESP32-S3-EYE. This board is specifically recommended for its integrated 2MP camera, 8MB PSRAM, 8MB Flash, and onboard microphone, which directly support all course assignments. It offers an excellent all-in-one solution for embedded AI development.
    • Justification: The ESP32-S3 is a powerful, cost-effective microcontroller with built-in Wi-Fi and Bluetooth, making it ideal for networked embedded AI applications. Its camera interface is crucial for computer vision assignments, and its I2S peripheral supports audio processing. The availability of numerous GPIO pins allows for flexible integration with various external sensors. This single platform can support all five assignments:
      • Assignment 1 (Fundamentals): Basic microcontroller programming, sensor data acquisition, and TensorFlow Lite Micro deployment.
      • Assignment 2 (Embedded Vision): Direct camera integration for image classification and model optimization.
      • Assignment 3 (Audio Processing): Microphone integration via I2S for keyword spotting.
      • Assignment 4 (Sensor Fusion): Ample GPIOs for connecting multiple external sensors (IMU, temperature/humidity).
      • Assignment 5 (Edge-Cloud Hybrid): Built-in Wi-Fi for seamless cloud communication.

2. Essential Peripherals and Accessories

  • Microphone Module (e.g., I2S MEMS Microphone like INMP441): While some ESP32-S3 boards might have an onboard microphone, an external I2S microphone module is recommended for better audio quality and dedicated audio processing assignments.
  • IMU Sensor (Inertial Measurement Unit, e.g., MPU-6050 or similar): For motion-based applications and anomaly detection in Assignment 4.
  • Temperature/Humidity Sensor (e.g., DHT11/DHT22 or BME280): For environmental data collection in Assignment 4.
  • Breadboard: For prototyping and easily connecting components.
  • Jumper Wires (Male-to-Male, Male-to-Female): For making connections on the breadboard and to the ESP32-S3 board.
  • Micro-USB to USB-A Cable: For programming and powering the ESP32-S3 board.
  • Small LEDs and Resistors: For basic output indicators and circuit building.

Software Requirements

1. Integrated Development Environments (IDEs) & Compilers

  • PlatformIO (VS Code Extension): Recommended for programming the ESP32-S3, offering a more robust and feature-rich environment than the Arduino IDE for complex projects.
  • Python 3.x: The primary programming language for machine learning development, data analysis, and host-side scripting.

2. Machine Learning Frameworks & Libraries

  • TensorFlow / Keras: For training and developing machine learning models on your computer.
  • TensorFlow Lite / TensorFlow Lite Micro: Essential for converting and deploying models to the ESP32-S3.
  • Python Libraries:
    • NumPy, Pandas: For data manipulation and analysis.
    • Matplotlib, Seaborn: For data visualization.
    • Scikit-learn: For traditional machine learning algorithms.
    • OpenCV (Python): For image processing tasks on the host computer (e.g., preparing datasets).
    • Pyserial: For serial communication with the ESP32-S3.
    • Requests (for Assignment 5): For making HTTP requests to cloud services.

3. Cloud Platform Account (Recommended for Assignment 5)

  • Google Cloud Platform (GCP), Amazon Web Services (AWS), or Microsoft Azure: Access to a cloud platform will be beneficial for Assignment 5, allowing students to experiment with cloud-based services for the edge-cloud hybrid system. Many platforms offer free tiers for basic usage.

Summary

By standardizing on the ESP32-S3 with camera module, students will gain in-depth experience with a powerful and widely used embedded platform in the AI and IoT space. This focused approach ensures that all practical assignments build upon a common hardware foundation, streamlining the learning process and allowing for deeper exploration of embedded AI concepts. Students are expected to acquire these components before the start of the course.
